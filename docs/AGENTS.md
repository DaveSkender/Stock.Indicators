# Documentation website development

This folder contains the VitePress documentation website for Stock Indicators.

## Quick start

```bash
# from /docs folder
pnpm install
pnpm run docs:dev
# Site will open at http://localhost:5173/
```

See .github/instructions/docs.instructions.md for complete VitePress development guidance, accessibility testing, and content guidelines.

See .github/instructions/markdown.instructions.md for markdown authoring standards.
