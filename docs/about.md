## About this library

**Stock Indicators for .NET** is an open-source C# library that transforms historical price quotes into technical indicators. The library provides industry-standard technical analysis indicators with a focus on accuracy, performance, and developer ergonomics.

### Project information

- **License:** [Apache 2.0](https://opensource.org/licenses/Apache-2.0)
- **Source code:** [GitHub repository](https://github.com/DaveSkender/Stock.Indicators)
- **Package:** [NuGet package](https://www.nuget.org/packages/Skender.Stock.Indicators)
- **Documentation:** [dotnet.stockindicators.dev](https://dotnet.stockindicators.dev)
- **Demo:** [Interactive charts](https://charts.stockindicators.dev)

### Python version

A Python port of this library is also available at [python.stockindicators.dev](https://python.stockindicators.dev).

## Chart generation

The indicator charts displayed throughout this documentation are generated using the following tools:

- **Chart library:** [Apache ECharts](https://echarts.apache.org/) - A powerful, interactive charting and visualization library
- **Chart data:** Generated from actual indicator calculations using this library
- **License:** Apache ECharts is licensed under the [Apache License 2.0](https://github.com/apache/echarts/blob/master/LICENSE)

### Attribution

We extend our gratitude to the Apache ECharts team for creating such an excellent open-source charting library that helps visualize technical indicators effectively.

## Technical details

### Target frameworks

The library targets actively supported Microsoft .NET platforms:

- .NET 10.0
- .NET 9.0
- .NET 8.0

The compiled library is [Common Language Specification (CLS) compliant](https://docs.microsoft.com/en-us/dotnet/standard/common-type-system) and can be used in other .NET languages.

### Quality standards

This library follows strict quality standards:

- Comprehensive unit test coverage
- Performance benchmarking with BenchmarkDotNet
- Mathematical precision validation against authoritative sources
- Continuous integration and automated testing

## Resources and references

Technical indicator formulas and methodologies are based on industry-standard references including:

- Published financial analysis textbooks
- Academic papers on technical analysis
- Established trading platforms and charting software
- Community-validated calculations

Specific references are documented in individual indicator pages where applicable.
